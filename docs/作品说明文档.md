# AI 驱动的 Roguelike 自动闯关游戏 —— 《命运裂隙》

## 一、作品名称

**命运裂隙（Rift of Fate）** —— 一款 AI 实时驱动的 Roguelike 无限闯关游戏。

## 二、核心玩法

游戏以"觉醒—技能构筑—无限闯关"为核心循环。开局时，AI 实时生成独特的觉醒体（含名称、被动和初始属性），归属八大方向：炎、霜、雷、影、命、圣、血、虚。同一"影"方向，本次可能是"暗影君王"（召唤分身），下次可能是"影之镰刀"（暴击回血但非暴击自伤）。觉醒体分双刃被动、纯正面被动和无被动三种类型。

闯关中每层触发5次事件。击杀怪物后，AI 根据觉醒类型和怪物等级实时生成限时技能，技能有回合或层数寿命，到期消失，迫使玩家持续战斗获取新技能。技能槽默认9个，最大18个。技能释放需消耗能量，形成资源管理策略。完整属性面板包含HP/ATK/DEF/能量/暴击率/暴击伤害/闪避率/幸运值，幸运值影响技能掉落、事件偏向、商人价格等。每5层触发里程碑事件。

## 三、AI 的核心作用

AI 在本项目中不是开发辅助工具，而是游戏运行时的核心引擎。游戏过程中，AI 通过 DeepSeek V4 Flash API 实时生成以下内容：独特的觉醒体（每局不同）、奇遇事件（携带觉醒上下文和玩家状态）、限时技能（匹配觉醒风格和怪物强度）、敌人配置（根据层数和难度动态调整）、里程碑事件叙事。每局游戏的觉醒体、事件、技能、敌人全部由 AI 即兴创造，确保每局体验独一无二。API 不可用时自动降级至预置内容池，保证游戏始终可玩。
