# AI-Roguelike-Game
# 命运裂隙 (Rift of Fate)

一款 AI 实时驱动的 Roguelike 无限闯关游戏。

## 项目结构

```
AI-Roguelike-Game/
├── README.md                    # 项目说明
├── docs/                        # 文档目录
│   ├── 游戏设计文档.md            # v0.4 完整游戏设计文档
│   ├── 作品说明文档.md            # 作品说明（<=500字）
│   └── AI使用方式说明.md          # AI 使用方式详细说明
├── game/                        # 游戏目录
│   ├── index.html               # 游戏主文件（单页 Web 应用）
│   └── data/                    # AI 生成的数据配置
│       ├── events.json          # 事件配置数据
│       └── difficulty.json      # 难度数值配置
└── prompts/                     # Prompt 对话日志
    └── 对话日志.md               # 完整 Prompt 对话记录
```

## 如何运行

直接在浏览器中打开 `game/index.html` 即可开始游戏。

推荐使用 Chrome / Edge / Firefox 等现代浏览器。

## 游戏简介

- **类型**：AI 实时驱动 Roguelike 无限闯关
- **核心机制**：觉醒体生成 -> 限时技能构筑 -> 能量管理 -> 策略闯关
- **觉醒系统**：AI 实时生成独特觉醒体，8大方向（炎/霜/雷/影/命/圣/血/虚），每局体验不同
- **技能系统**：AI 根据觉醒类型和怪物等级实时生成限时技能，到期消失，保持构筑流动性
- **能量系统**：技能释放需消耗能量，形成资源管理策略
- **属性面板**：HP/ATK/DEF/能量/暴击率/暴击伤害/闪避率/幸运值
- **难度**：四档可选（简单/普通/困难/地狱），影响敌人强度、技能寿命、奖励系数
- **事件**：AI 实时生成奇遇事件，携带觉醒上下文和玩家状态，每局独一无二
- **里程碑**：每5层触发固定事件，保证技能获取节奏
- **三套等级**：层数（进度）/ 玩家等级（属性成长）/ 怪物等级（强度与掉落）

## AI 技术说明

本项目的核心特色是 **AI 实时集成**——AI 不是开发辅助工具，而是游戏运行时的内容引擎。

- **AI 引擎**：DeepSeek V4 Flash API，低延迟实时调用
- **实时生成内容**：觉醒体、奇遇事件、限时技能、敌人配置、里程碑事件叙事
- **上下文注入**：每次 AI 调用携带完整玩家状态（觉醒信息、属性面板、技能列表、幸运值等）
- **安全机制**：属性上下限约束、平衡评分算法、JSON 校验、自动裁剪
- **降级策略**：API 不可用时自动回退预置内容池，保证游戏始终可玩

详细说明请参阅 `docs/AI使用方式说明.md`。

## 技术栈

- 纯前端：HTML5 + CSS3 + Vanilla JavaScript
- 无外部依赖，单文件可运行
- AI 引擎：DeepSeek V4 Flash API（运行时实时调用）
- 数据格式：JSON（AI 输入输出）
